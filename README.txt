Guia HTML - IA para donos que viraram gargalo do proprio negocio

Arquivos:
- index.html: versao pronta para publicar em GitHub Pages, Vercel, Netlify ou qualquer hospedagem estatica.
- guia-ia-donos-gargalo.html: copia com nome descritivo do guia.

Observacao:
O guia e um HTML autonomo. Logo, favicon, estilos e scripts estao embutidos no proprio arquivo.
Para publicar, suba o arquivo index.html na raiz da hospedagem.
